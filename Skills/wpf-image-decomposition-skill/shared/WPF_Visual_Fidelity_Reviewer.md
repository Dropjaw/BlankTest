# Shared Skill — WPF Visual Fidelity Reviewer

## Purpose

Compare a mock image and implemented WPF screenshot to identify mismatches and generate precise corrective guidance.

## Review Categories

### 1. Shell Fidelity

Check:
- window chrome
- top bar
- side navigation
- status bar
- active navigation state

### 2. Layout Fidelity

Check:
- rows/columns
- widths/heights
- alignment
- panel positioning
- scroll boundaries

### 3. Component Fidelity

Check:
- cards
- table
- buttons
- search
- filters
- badges
- quick actions
- details panel

### 4. Density Fidelity

Check:
- margins
- padding
- gaps
- row heights
- card heights
- icon sizes

### 5. Typography Fidelity

Check:
- font size
- weight
- colour
- alignment
- wrapping/trimming

### 6. Colour Fidelity

Check:
- background
- card surface
- border
- primary accent
- status colours
- selected state

### 7. Interaction Fidelity

Check:
- selection
- command enabling
- hover/focus
- search/filter
- pagination
- refresh
- navigation

### 8. Accessibility

Check:
- keyboard order
- focus states
- accessible names
- non-colour status indicators
- contrast

## Severity

- P0: functional/layout blocker
- P1: obvious mismatch
- P2: polish issue
- P3: optional refinement

## Output

- fidelity score
- discrepancy list
- prioritised fix plan
- fix prompt
